# About Me

Data Analyst with a background in software development, I work across the full data lifecycle, including data identification, collection, cleaning, statistical analysis, financial process automation, dynamic reporting, ETL workflows, and business intelligence.

I develop optimization solutions, automate financial and operational processes, and create dashboards that support strategic decision-making. My experience combines software engineering, cloud technologies, databases, and data visualization to transform complex data into valuable business insights.

Beyond technical expertise, I value analytical thinking, continuous learning, effective communication, teamwork, and delivering high-quality solutions.
